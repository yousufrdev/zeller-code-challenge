## Getting Started

Follow these steps to run the project from scratch:

### 1. Clone the repository

```bash
git clone <repository-url>
cd <repository-folder>
```

Replace `<repository-url>` with your GitHub repository URL.

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm start
```

### 4. Run tests

```bash
npm test
```
